# 遥操作机械手项目

## 项目简介
远程同步手部动作的机械手——左手戴手套(弯曲传感器 + IMU)采集动作,通过 ESP32-S3 无线传输给右手的 5 指机械手,实现远程同步。

## 硬件
- 主控:ESP32-S3-DevKitC (N16R8) × 2
- 6 × LX-16A 总线舵机
- 5 × 弯曲传感器
- 1 × BNO055 IMU
- 1 × 6S 航模锂电池 + BEC 降压模块
- 3D 打印件(PETG)
- M2 螺丝 + 624ZZ 微型轴承
- 钓鱼线 0.8mm(拉线传动)

## 通信协议(主从)

帧结构 32 字节:
- [0-9]   flex[5]   uint16_t × 5   弯曲传感器
- [10-13] quat[0]   float (w)      IMU 四元数
- [14-17] quat[1]   float (x)
- [18-21] quat[2]   float (y)
- [22-25] quat[3]   float (z)
- [26-27] seq       uint16_t       序列号
- [28-31] reserved  uint8_t × 4    保留

发送周期:10ms

## 接线备忘

### 弯曲传感器(分压电路)
```
3.3V ── [10kΩ 上拉] ──┬── ESP32 GPIO 1 (ADC1_CH0)
                       └── 弯曲传感器一端
                       └── 弯曲传感器另一端 ── GND
```

### LX-16A 舵机(1kΩ 电阻方案,单舵机够用)
```
ESP32-S3 GPIO 17 (U1TXD) ── 1kΩ ──┬──→ LX-16A 信号线(橙)
ESP32-S3 GPIO 18 (U1RXD) ─────────┘
ESP32-S3 GND  ────────────────────→ LX-16A GND(黑)
7.4V +  ──────────────────────────→ LX-16A V+(红)
7.4V -  ──────────────────────────→ LX-16A GND(黑,共地)
```

**原理**:LX-16A 是单线半双工总线(1 根信号线收/发共用)。
ESP32 UART 不发数据时,TX 引脚是**高阻态**(相当于断开),
所以 1 个 1kΩ 电阻 + 软件时序切换就能搞定,
**不需要专用控制板**。

**多舵机(5+)升级**:加 1 个三极管(2N7000)做 DIR 控制,
或加 SP485 模块(10 元)。阶段 4 才需要。

### BNO055 IMU
```
BNO055 VIN → 3.3V
BNO055 GND → GND
BNO055 SDA → ESP32 GPIO 8 (I2C SDA)
BNO055 SCL → ESP32 GPIO 9 (I2C SCL)
```

## 安全提醒
- 电池千万别短路——会起火爆炸
- 舵机堵转时间别超过 5 秒——会烧舵机
- 调试时把机械手固定在桌上——别让它飞出来砸你
- 戴手套时注意拉线——别缠到手指上
- 每次通电前检查接线——接错一根可能烧一片
- 家里备个灭火器

## 卡住了怎么办

1. **接线错** → 重看原理图,万用表量通断
2. **代码 bug** → Arduino IDE 串口监视器看打印
3. **库没装对** → 重装库,GitHub 找原版
