# YD-ESP32-S3-N16R8 板子引脚备忘

来源：`ESP32-S3 资料\ESP32-S3 原理图.pdf` (YD-ESP32-S3-COREBOARD V1.4)
型号：YD-ESP32-S3-N16R8（源地工作室，仿 ESP32-S3-DevKitC-1 布局）

## 板载 LED

| 项 | 值 |
|---|---|
| 类型 | WS2812B (XL-5050RGBC) |
| 数据脚 | **GPIO 48** ⚠️ 不是官方 38 |
| 颜色顺序 | GRB |
| 库 | FastLED（Arduino 库管理器搜 "FastLED"）|

测试代码：`board-test\02_led_finder\02_led_finder.ino`

## 板载按键

| 按键 | 接哪个 GPIO |
|---|---|
| BOOT | GPIO 0 |
| RST | CHIP_PU (EN) |

## USB 接口

| 接口 | 类型 |
|---|---|
| 左 Type-C | USB OTG 直连 ESP32-S3 原生 USB |
| 右 Type-C | USB 转串口（CH343P 芯片）→ UART0 |

Win10/11 一般自带 CH343 驱动。如果不识别，去乐鑫或南京沁恒官网下驱动。

## 已确认有效的固件参数（Arduino IDE 工具菜单）

| 项 | 值 |
|---|---|
| Board | ESP32S3 Dev Module |
| USB CDC On Boot | Enabled |
| CPU Frequency | 240MHz (WiFi) |
| Flash Mode | QIO |
| Flash Size | 16MB (128Mb) |
| Partition Scheme | Default 4MB with spiffs |
| PSRAM | OPI PSRAM |
| Upload Speed | 921600（如果失败改 115200）|

## MicroPython 备用方案

YD 为这块板子定制了 MicroPython 固件：
`ESP32-S3 资料\固件\YD-ESP32-S3-N16R8-MPY-V1.1.bin`

刷入工具：`ESP32-S3 资料\下载工具\flash_download_tool_3.9.3\flash_download_tool_3.9.3.exe`
刷入方法：选 esp32s3，地址 0x0，勾选要刷的文件，点 START，按住 BOOT 再按 RST 进入下载模式。

如果哪天 Arduino 玩腻了想换 MicroPython，可以直接用这个。
